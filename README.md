# Semi_Network_Pytorch

### Files in the folder

- `data/`:
  - `testing/`:
    - images
  - `training/`:
    - images

- `code/`: implementations of Semi_Net

### Required packages

The code has been tested running under Python 3.6, with the following packages installed (along with their dependencies):

- numpy == 1.14.5
- scipy == 1.5.4
- pytorch == 1.7.0
